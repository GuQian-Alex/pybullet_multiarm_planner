# pybullet_multiarm_planner

`pybullet_multiarm_planner` is a motion planning project that integrates traditional planning algorithms with a PyBullet simulation environment for multi-arm robot planning.

The project uses PyBullet for robot simulation and collision checking.

It uses both hand-write traditional planning algorithms and OMPL algorithms for benchmarking.

It is designed for experimenting with single-arm and multi-arm motion planning problems in a simulated environment.

## Features

- PyBullet-based robot simulation
- Multi-arm motion planning
- Traditional motion planning algorithms
- OMPL integrated for benchmarking
